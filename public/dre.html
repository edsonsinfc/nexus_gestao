💼 DRE Módulo API

A Demonstração do Resultado do Exercício (DRE) é um relatório contábil que mostra o desempenho financeiro da empresa em um período específico, apresentando receitas, custos, despesas e o resultado líquido.

📡 Endpoints Disponíveis

GET /api/dre?data_inicio=2025-01-01&data_fim=2025-01-31
Parâmetros: data_inicio, data_fim
Retorna: DRE completo com 12 seções (Receita Bruta, CMV, Lucro Bruto, etc.)
GET /api/dre/comparar?periodo1_inicio=...&periodo1_fim=...&periodo2_inicio=...&periodo2_fim=...
Retorna: Comparação entre dois períodos com variações percentuais
GET /api/dre/categorias
Retorna: Lista de categorias financeiras disponíveis

📊 Estrutura do DRE

  • ✅ Receita Bruta
  • ✅ (-) Deduções da Receita
  • ✅ (=) Receita Líquida
  • ✅ (-) CMV (Custo da Mercadoria Vendida)
  • ✅ (=) Lucro Bruto
  • ✅ (-) Despesas Operacionais
  • ✅ (=) Lucro Operacional
  • ✅ (+/-) Resultado Financeiro
  • ✅ (=) Resultado Antes dos Impostos
  • ✅ (-) Impostos sobre o Lucro
  • ✅ (=) Resultado Líquido
  • 📈 Margens: Bruta, Operacional e Líquida

🔧 Como Usar

Integre via JavaScript/Fetch API:

const token = localStorage.getItem('token');
const response = await fetch('/api/dre?data_inicio=2025-01-01&data_fim=2025-01-31', {
  headers: { 'Authorization': 'Bearer ' + token }
});
const data = await response.json();
console.log(data.dre);
